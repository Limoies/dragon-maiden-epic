**实际执行**：✅ 已完成

docs/
├── worldview/    # 世界观总览
├── mythology/    # 神话与起源
├── geography/    # 地理（含染血草原子目录）
├── characters/   # 角色
├── history/      # 历史纪元
├── factions/     # 组织与国度

## 附录：图片插入方法

### 存放位置

将图片放入 `docs/public/` 下，可按分类建子文件夹：

```
docs/public/
├── characters/    ← 角色立绘
├── map/           ← 地图
└── bestiary/      ← 图鉴
```

### 引用方式

在 Markdown 中用 `![]()` 语法：

```markdown
![莉娅·蔷薇鸟](/characters/liah.png)
```

图片默认居中显示，响应式适配屏幕宽度。

### 注意事项

| 问题 | 原因 | 正确做法 |
|:----|:----|:--------|
| ❌ `<div>` + Markdown 混用报错 | Vue 编译器无法解析 HTML 块内的 Markdown 语法 | 在 `<div>` 内用纯 HTML（`<img>`、`<ul>`、`<strong>`）或干脆不用 HTML 块 |
| ❌ 图片路径不对 | 路径写错或放错位置 | 统一放 `public/` 下，引用路径以 `/` 开头 |

## 附录：常见问题

| 问题 | 原因 | 解决 |
|:----|:----|:-----|
| 页面 404 | 文件有 UTF-8 BOM 头 | 移除 BOM |
| VitePress 启动失败 | 版本过旧（alpha）与 Node v24 不兼容 | 升级到 1.6.4 |
| `hasInjectionContext` 报错 | Vue 3.2 过旧 | 升级到 3.5 |
| `Invalid vnode type` | 组件注册方式不对 | 改用 Layout.vue import 而非 app.component |
| 粒子被挡住 | canvas z-index 太低 | 设为 z-index: 5 |
| 点击涟漪无反应 | canvas `pointer-events: none` | 改监听 window 事件 |
