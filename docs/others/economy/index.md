# 货币体系

## 通用货币层级

| 层级 | 名称 | 汇率 |
|:---:|:----|:----:|
| 1 | **钢珠** | 10钢珠 = 1灼铜（单向） |
| 2 | **灼铜** | 20灼铜 = 1萤银 |
| 3 | **萤银** | 20萤银 = 1霜银 |
| 4 | **霜银** | 50霜银 = 1鎏金 |
| 5 | **鎏金** | 最高级货币 |

> 备注：1钢珠约等于现实五角钱。

## 各货币详解

### 钢珠

直径约1.2厘米，通体温润的暗红色完美球体，无法伪造。含于舌下会尝到极淡的铁锈甜味。

来源："熔炉内的司康顿尔"一类基础仪式魔法的伴生产物。无人为获利而铸造钢珠，只有使用仪式魔法时积攒下来的副产物。

### 灼铜

赤红色铜币，表面压印联合商会的纹章。铸币权由三大主城内的联合商会垄断铸造。

### 萤银

银白色币面，印有主城的家族纹章，铸造时掺入龙萤粉，具有随角度变化的纯白荧光。铸币权只由三大主城的城主掌有。

### 霜银

币面呈冷白色，覆盖一层寒霜，取自北方山脉永冻层下的寒髓矿。握于掌心前三十秒冰冷刺骨，三十秒后与普通银币无异。

铸币权仅由曦日王庭垄断，原材料亦为王庭专采。

### 鎏金

无法直接铸造，只能通过司康顿尔谱系下的最高级仪式魔法与熔龙火山特殊环境，使用熔龙火山内珍稀矿物得到。
