# 武器特质

> *"在这个世界观里，所有武器都是法杖，法杖的类型也千姿百态。可以是长剑，可以是弓，可以是大盾……以下特质适用于你能想象的一切武器——只是表现形式不同。"*

## 武器的六维

### 1. 镌刻魔法（存储）

法杖上镌刻好特定魔法的符文，可以直接施加魔力调用符文进行施法，省下吟唱时间。一位普通魔女吟唱只能放一环火球术，但借助镌刻了二环火球炮的法杖，就能成为人形炮台。

### 2. 辅助施法（增益）

借助法杖可以更轻松地感知元素流向，辅助吟唱更高级的法术。不同的武器有不同增益效果——某家族流传的剑可能通过血脉共鸣增强使用者的体质。

### 3. 特殊技能（核心）

有名的武器都有着一些核心技，通过灌输大量魔力或一定代价爆发出来。

### 4. 暴揍敌人（强度）

回归武器的本源。强度越高越说明武器质量上乘。

### 5. 使用要求（操纵）

好的武器需要好的主人。越难使用的武器越珍贵。当然，与血脉也有关。

### 6. 独有特质（？）

它不像前五维对武器性能的评价，更像是用一句话描绘它的传说，定义它的"性格"。
