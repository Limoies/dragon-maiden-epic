# 猎人公会

## 信条

*   不杀同道，纠纷由公会仲裁。
*   不毁龙骨，龙巢的骨骼是庇护所，是所有人的财产。
*   不拒委托，符合星级的任务必须接。
*   不弃同伴，抛弃队友者视为自动放弃猎人资格。

## 龙心灯

龙心灯是公会的精神象征。它是一簇永不熄灭的幽蓝冷焰，由古龙残留的龙髓与魔素反应生成。猎人远行前会在灯前静立，自问：“我准备好了吗？我能活着回来吗？”

若有猎人未能归来，其搭档会在灯下放一枚灼铜币。公会定期将这些铜币熔铸成铜牌，刻上逝者的名字。
