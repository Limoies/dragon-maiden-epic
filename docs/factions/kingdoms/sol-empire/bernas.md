# 贝尔纳斯

> *"荆棘中开花，灰烬中重生。"*

贝尔纳斯位于夏伊斯岛中东部偏南，伊河流域，是曦日帝国的第三主城。城郊的蔷薇领是蔷薇鸟家族的封地，以盛产蔷薇酒闻名。

### 蔷薇鸟归来！万人空巷

昨日上午九时，**莱昂·蔷薇鸟伯爵**在离开十五年后，正式回归蔷薇领，恢复爵位并签署《贝尔纳斯共治协议》。

上万民众涌上街头，挥舞着银底血色蔷薇的旗帜。有位老农激动地把自家养的牦龙犊扛上肩头（据说那头犊子一脸茫然，但很配合地"哞"了一声），还有酒馆老板当场宣布"今日蔷薇酒八折"……

莱昂伯爵登台时，没有慷慨激昂的演讲。他沉默了很久，才开口：

*"我回来，不是来征税的，是来问你们——这些年，过得怎么样？"*

台下先是一静，然后爆发出震耳欲聋的欢呼。

莱昂侧身，向台下伸出手——两个少女被推上台来。一个是面无表情但耳尖微红的猎人少女，另一个正拼命把裙摆上干涸的龙血往身后藏。

*"我的女儿。莉娅，莉姆。"*

---

### 前代掌权者结局

前蔷薇领代掌权者**瓦尔德·曦日**已被移交王庭，押返王都候审。

一位老农说："他终于走了。"旁边的大婶接话："是啊。走了就好。不提了。"
