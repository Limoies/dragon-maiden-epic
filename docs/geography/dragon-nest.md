# 龙巢小镇

> *"肋骨为墙，环廊为街。古龙的遗骨，撑起了猎人的家。"*

## 概述

龙巢小镇是建在一具远古巨龙遗骸上的猎人自治聚落。巨大的龙骨构成了小镇的主体框架——肋骨化作围墙与拱门，脊椎成为贯穿全镇的主干道。

## 龙心灯

小镇的核心是一盏永不熄灭的「龙心灯」。传说这是古龙陨落时心脏残留的能量结晶，猎人们相信只要灯火不灭，小镇便永受庇护。

## 猎人文化

作为边陲前哨，龙巢小镇是猎人的大本营。镇上的每一家酒馆都挂满了战利品，每一个老猎人都有一肚子说不完的故事。
