# 深渊与腐败之地

> *"这不是一道伤疤，而是一张永不闭合的嘴——它永远在吞噬，也永远在呕吐。"*

## 概述

深渊位于夏伊斯岛西南部，外形如同一头侧卧古龙的下颚处被撕裂的创口。这里是**原初魔女被分食之地**，腐败的源头。

## 腐败之海

从深渊裂缝中，腐败如同黑色的血液般不断渗出，染黑了周围的海域。靠近此地的生物——无论是龙族还是人类——都会逐渐被腐败侵蚀。

## 禁区

帝国将深渊方圆百里划为绝对禁区。但仍有不知死活的冒险者试图深入其中，寻找传说中残留的「神之遗物」。绝大多数人再也没有回来。
