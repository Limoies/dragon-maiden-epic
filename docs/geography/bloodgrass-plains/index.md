# 染血草原

> *"草原有四个风：煦风吹生，飙风吹死，凝风吹忘，裂风吹醒。"*

## 概况

染血草原位于夏伊斯岛东南部，平均海拔约2800-3200米的高原台地。传说远古神战之时，神明之血洒遍这片高原，草叶从此染上了永不褪去的暗红。

## 鸟龙目的天空剧场

### 气流剧场：山与崖锻造的飞行圣殿

染血草原东部边缘矗立着一道绵延四百余里的断崖——**斩风崖**，平均落差800-1200米，局部段落超过一千五百米。

来自东方海洋的暖湿气流被断崖迎面截击，被迫抬升，形成永久性的上升气流带与湍流区。崖壁自身的凹凸、褶皱、崩塌形成的岩柱群，又将气流切割成无数股方向、温度、湿度各异的乱流——对于飞行生物而言，这是天然的飞行训练场与狩猎场。

鸟龙目，尤其是风翎枭龙，正是以驾驭乱流为天性的物种。

### 热力引擎：曦日辐射的馈赠

海拔三千米的高原，白天草原地表吸收大量热能，形成强劲的热力上升气流；入夜，地表迅速冷却，形成稳定的滑翔层。这为鸟龙目提供了全天候的飞行能源。

## 地形分层（自西向东）

## 四季之风

- **煦风季（6-8月）**：温暖、多雾、湿润。幼龙鸟破壳，草原最慈悲的时刻。
- **飙风季（9-11月）**：风暴之季。斩风崖前千鸟齐翔，游牧民称其为"天空的战场"。
- **凝风季（12-2月）**：风止、草枯、兽眠。牧民围炉，万物等春。
- **裂风季（3-5月）**：东风与西风在草原上空撕扯，龙血草尖刺破冻土。最残酷的春天。
