# 夏伊·帕莉薇尔

> *"三万千米的高空，她如流星般坠落。陨石撕开风暴，熔岩在她周身环绕。她睁开眼，忘记了过去，却听见了自己的名字。"*

## 基本信息

- **身份**：第三魔女
- **种族**：魔女
- **特征**：银发、失忆、胸前佩戴净化吊坠
- **能力**：龙语魔法（"青空之风"）、符文魔法（火）、腐败净化

## 背景

夏伊降临在染血草原上，巨大的动静惊动了整个高原。当她从陨石坑中站起时，已经失去了所有的记忆——只记得自己的名字。

她是预言中第三位魔女，唯一能净化腐败的存在。她的抉择将决定世界的终局。

## 染血草原篇

夏伊的降临被莉莉丝（魔女教会圣女）和猎人姐妹注意到。在染血草原上，她遭遇了被腐败侵蚀的风翎枭龙，在危急时刻本能地挡下攻击，用莉姆的法杖刺穿枭龙的弱点——同时，胸前的吊坠牵引了一部分血脉，净化了腐败。

这个细节被莉莉丝看在眼里，内心动摇了一瞬：*既然她能净化腐败，那她能否成为这个世界的救世主呢？*

在莉莉丝的暗中策划和猎人姐妹的帮助下，夏伊穿越染血草原，抵达龙巢小镇，开始了她的旅程……
