# 风翎枭龙

> *"它的翼划过天空，如同风在书写一首无人能懂的诗。"*

## 基本信息

| 项目 | 内容 |
|:----|:------|
| 分类 | 鸟龙目 · 枭龙科 · 风翎属 |
| 栖息地 | 染血草原及周边高崖、风蚀峡谷 |
| 生态定位 | 空中顶级掠食者 |
| 别称 | 苍穹哨兵、风之祭司、青空执法者 |

## 生理结构

### 体型
- 站立高约4.5米，体长（含尾）约9米
- 翼展8-10米，翼形宽大呈圆弧状
- 体重约1.2-1.8吨，骨骼中空轻质

### 羽毛与体色
- 淡青蓝色绒羽打底，覆盖银白色流线型羽纹，阳光下呈现"青空流转"效果
- 颈后、翼缘、尾尖生长特殊透明**风感翎羽**，能敏锐捕捉气流变化
- 胸腹、翼关节处覆盖细小青灰色龙鳞

### 头部特征
- 猫头鹰式面盘，增强声波接收能力（可感知千米外猎物心跳）
- 黄金竖瞳，夜视极强，自带低阶龙威压迫
- 喙呈钩状，边缘有细密锯齿；喉部结构特殊，可发出多重频率鸣叫

## 能力

风翎枭龙能够驾驭斩风崖的乱流，在狂风中盘旋、俯冲、急转，如同海豚在浪涛中嬉戏。有经验的猎人能从其飞行姿态判断当日的气流强度——"三旋回以上，明日必有大风"。
