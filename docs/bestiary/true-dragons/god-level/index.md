# 神阶真龙种

## 定义

神阶真龙种是真龙种的最高位阶。祂们持有**本源龙语**——这是龙语谱系的根源，所有君王阶的直系分支与君主阶的变种分支，皆从本源分化而来。

一位神阶真龙种可以同时持有多个本源龙语。

## 已知神阶

### 北方的神
龙族群岛北方诞生的神阶真龙种。同时持有“日”与“火”两项本源龙语（可能更多）。第二纪元神战的终结者，用自己的权威压制了君王间的灭世战争，确立了第三纪元的基本秩序。

### 南方的死神
稍晚于北方神明诞生的第二位神阶真龙种。只持有“死”一项本源龙语。祂的诞生意味着龙族群岛从此进入南北并立的神明格局。
