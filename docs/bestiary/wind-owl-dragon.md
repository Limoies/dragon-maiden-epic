# 风翎枭龙（亚龙·低阶）

## 分类
*   **目**：鸟龙目
*   **科**：枭龙科
*   **别称**：苍穹哨兵、风之祭司

## 生态特征

站立高约4.5米，翼展8-10米。淡青蓝色绒羽，猫头鹰式面盘，金黄色的瞳孔带有低阶龙威。

它们利用斩风崖的永久上升气流狩猎，是染血草原的天空顶级掠食者。龙语法纹本能地刻印在其翼膜上，能吟唱序列4的弱化版“风暴之息”。

## 腐败侵蚀

被腐败侵蚀后，瞳孔会染上墨色，能短暂施展序列5的弱化版“天穹之幕”领域。这会导致它们失控，攻击一切。夏伊的第一次净化，便是对一只被腐败侵蚀的风翎枭龙。
