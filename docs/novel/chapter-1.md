# 第一章 · 自坠星降下的少女

> *"高天之上的神明啊，我向您祈盼。愿您能佑我不度困苦、不失所愿……"*
>
> ——序

三万千米的高空之上……深黑色的夜幕宛若浪潮一般吞没着整个天空，永世不变。

但一道炽烈的红线划破了这份宁静。

一颗流星正以极快的速度坠落而来，裹挟着炽热又庞大的烈焰"轰隆隆"地撕扯开了大气，紧接着向天幕之下那翻滚着咆哮着的厚重风暴坠去。

仿佛如巨龙的吼声震荡开来，先前没有一丝一毫损伤的陨石此刻正疯狂解体着。

极致的冲击力砸向、冲刷着咆哮般的风暴，疯狂又持续的碰撞擦出了巨型的粒子乱流，在尾后拖拽出一条绚烂的"红线"。

陨石毫不停歇地向地面冲去，最终在仅余千分之一的体积下穿越那厚重的云层。

无尽遥远的彼端，在云层下方，散发着无尽光芒的烈日正照耀着这片大陆。

大陆？不，是一片面积无穷广袤的连绵群岛，平铺在深蓝色表面，有些串联在一起，绵延至那片蓝色的彼端。有些则分离开来，中间隔着一道狭长的裂缝，永无止境地吞噬着周围一大团蓝色。

陨石疯狂地冲向地面，景色也越来越清晰、接近。

最终，在一片大草原上，身披苔藓状鳞片的龙兽正没在繁茂的野草中啃食着，脏污的鳞片上映出了那灭世的火光。

紧接着一些抬起头，惊惧地望向天空。

它们见到了此生第一次见，同时也是最后一次见到的景象……

一个黑点从天空的彼端坠下，然而刹那间就划过了整个天幕，在名为天空的画布上绘下了一条火红的线。
