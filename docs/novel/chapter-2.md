# 第二章 · 风翎枭龙

夏伊拢了拢身上的长袍，站起身来，拍了拍身上的草灰，恍惚地看向前方。

刚刚的黑袍少女早已退至一旁，越过黑袍少女，在她身后，一位身穿白袍的老者正威严地坐在一匹有着赤色鬃毛的大马上。

更往后，一望无际的草原上，三四匹赤鬃马牵着马车，零零散散的停在一旁。

仿佛述说着横跨草原之途的艰辛，马车的外表大部分都带着严重的划痕或是凹陷痕迹，但整体还算完整。原木色的外表上雕刻着一些鎏金色的纹路，在黄昏下熠熠生辉。车厢后部，一张巨大的黑色帆布覆盖其上。

穿铁甲的士兵或骑在马上或倚着马车休息着，零零散散接近十几人，但神情都十分疲惫和呆滞……

踏在草地上才有的厚实脚步声从前方传来。

视线骤然收近，那位身穿白袍的老者不知何时已缓缓走近，草原上的大风吹拂着长袍随风飘动，掀起猎猎风声。他的脸上挂着慈祥的笑容却用一双狂热而浑浊的眼神死死盯着少女，强大的气场压得她下意识地后退一步。

等等等下，这是看上去七十多岁的老头吗？呃，我是应该说明还是直接逃跑……

“等，等下！那个……”

正当夏伊语无伦次地想要解释时，“扑通”一声，白袍老者突然单膝跪在了少女的面前。

“赞美神明梅瑟丝尔！”

“吾名赞恩。啊！您，您就是我一直以来苦苦寻觅的神女大人啊！”

潜意识已经准备好开溜的夏伊仿佛突然被刚见的大马以80km/h的时速创到了脑袋，后退的小脚骤然停住，墨色的瞳孔里充满大大的疑惑与警惕，她小嘴张了张，下意识地吐出一个词。

“啊？”

这不对吧，谁会对从陨坑里飞出来的女生称呼苦苦追寻的神女啊！等等……貌似迥于这个世界的常识、从陨坑中苏醒，那我的过去……

少女身体猛地一弯，一手拢紧过于宽大的长袍，一手死死扯着自己的头发。

脑海中一道道强烈的灼烧感传来，疯狂地燃烧着渴望着吞噬些什么……夏伊死死咬着自己的嘴唇，不让自己喊出声来。

这时，一阵比刚才强风更为强劲且杂乱无序的狂风袭来，缓解了夏伊的灼烧感，也打断了老者准备做的举动。

白袍老者赞恩的眼眸染上一丝疑惑，站起身来望向四周，在休息的骑兵也注意到了异常，纷纷警惕起来。

早已退到他身后的黑袍少女不久前已从长袍中抽出四颗通红的钢珠，一把撒向草地，同时低吟道。

“我，祈求司康顿尔之名！”

少女又从衣袖中抖出一把磁石碎屑，它们有规律地如水波般振动着。

“碾碎虚妄之壁，熔断蒙昧之锁——”

一声轻哼，接着一滴鲜血滴落在钢珠上，瞬间钢珠如入熔炉般沸腾起来，不断升华又凝华，在磁石碎屑波中凝结成千奇百怪的形状。

“令暗影显于钢水，令呼吸响彻焦土！”

少女眨了眨眼，大地瞬间变为灰白色的焦土，前方的老者和那个正紧咬嘴唇貌似来历不明的女生身边浮现一层炽红色的余烬轮廓。

她抬头向上看，透过数千米的云层，一道同样带着炽红色的轮廓浮现在她瞳孔中……

“低阶亚龙种、鸟龙目，风翎枭龙，空中！”

“铁甲队，列阵——，保护马车和祭品！”

祭品？

夏伊刚站起身来就听到黑袍少女清脆的声音和老者雄浑的吼声，还没等反应过来，一声响彻天幕的风哨声贯穿所有人的耳膜。

狂暴的风一并袭来，杂乱暴躁地吹着她的长袍“呼呼”作响。

等下……什么玩意？

夏伊迎着风艰难的转身，长袍突然被人往后拽了一把，整个人也被拽着三步并两步往后退去。

几乎是同时，高空中一道黑影裹挟着强风降落在她的眼前。那是一只体型庞大的鸟龙目，它张开双翅，巨幅的翼展将近8米。淡蓝色的绒毛夹杂着白色羽纹，在黄昏下闪烁着微光。圆圆的头上一双金黄色的瞳孔死死地俯视着夏伊，就好像……

一只巨型的猫头鹰？

下一瞬间，一圈十字形的大范围巨型风刃朝她和四周斩去。

无形的风刃携着破空声沿着泥地划出沟壑，卷起的灰尘如墙一般瞬息间扑面而来，下一秒，风刃席卷到夏伊眼前。

“刺啦——”一道破空声传来，一柄血色的长枪从中间刺穿了这道风刃。

夏伊偏头看去，眼中映出一位约莫19岁的少女。

淡灰的及腰长发被风高高卷起，额头两侧好似镶着饰品一样的深红色龙角。她身穿着有金属质感的漆黑胸甲，搭着一件洁白无瑕的银白色束腰短裙。

“莉娅。”瞥了夏伊一眼，她淡淡地说道，随即往前踏了一步。

“既然莉莉丝相信你……看好我妹。”

没等夏伊理解这句话，下一瞬间，她握紧那把血色长枪，像箭一样迎着风刃朝那只风羽枭龙冲去。

“姐姐，我叫莉姆，你叫什么？”

“啊……我，我叫夏伊。”感觉到有人扯着自己的衣服，夏伊回头看去。

好可爱……应该是一位15岁左右的少女，为什么手握着一个巨大的红色权杖——等下，来不及想那么多了。

狂风呼啸着滚过，前方纤细的身影如蔷薇花瓣一般轻盈，恰能在闪转腾挪间随手用长枪刺穿每一道风刃。

“莉姆，你，有武器吗？我想去帮你姐忙。”

“谢谢你的好意。不过，姐姐大人——可是很厉害的。”

一轮攻势完，四周的土壤早已被风刃切割得四分五裂，砂土纷飞。

前方的风翎枭龙似乎打算切换攻击方式，淡青色的巨大绒翼扑扇着准备起飞。

下一瞬间，红芒于枪尖绽放，莉娅动了。

没有任何多余的动作，侧身闪过掀起的风刃，脚下一踏，瞬间到风翎枭龙面前。

突刺，收枪，接着甩出一轮血花。

夏伊只来得及看清眼前一道红色的残影穿刺而过，接着一道殷红的血雾和痛苦尖锐的龙吟爆发开来。

杂乱无章的风以风翎枭龙为中心爆发开来，头发胡乱地拍打在夏伊的脸上，强风刮得她连眼睛都很难睁开。

四周泥沙纷飞，原本挺立的龙血草也被风刮得东倒西歪，甚至无法用混乱来形容了。

但那殷红的血雾却仿佛无视了狂风的影响，虽有浮动，却死死地笼罩在风翎枭龙的周身。

模糊中，夏伊好像看见风翎枭龙的前喙朝着血雾中的人影狠狠啄了下去。

“莉娅！”迎着风，夏伊大喊道。

但在被锋利的前喙啄到的那一刻，人影又消散开来化为一片血雾。

“姐姐没事！这是蔷薇龙语的一类魔法——幻象。”在她身旁的莉姆解释道，尽管声音在风的影响下忽大忽小。

“蔷薇龙语？”

“嗯……大概就是……”少女没做回答，而是将食指于巨大权杖的尖端上轻轻划过，尖锐的顶端割伤了莉姆柔嫩的皮肤，一丝血雾弥漫开来。

“蔷薇。”

一个晦涩、拗口，神秘而又动听的单词从少女的口中吐出。
