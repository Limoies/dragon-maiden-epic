# 第三章 · 腐败

刹那间，笼罩周围的血雾沸腾起来，渗透着钻进风翎枭龙的伤口，接着扭曲着从龙鳞下扎根生长出一株株猩红色的荆棘，绽放开一株株妖艳美丽的蔷薇。

但下一秒，无数细微的风刃又将蔷薇切割得凋零不堪。风翎枭龙咆哮一声，正准备扑扇着双翼飞起，随即又被莉娅凌厉的攻击牵制住。

莉娅的身形在血雾中穿梭，每一次闪现都带起一道龙血飞溅，但风翎枭龙的反扑愈发狂暴。

不会还要来一次吧，很痛的呀。

莉姆脸色一僵，面色扭曲地再次用手指颤抖着移到杖顶，轻轻一划。

无数猩红的花蕊再次于风翎枭龙上绽放，尽管顷刻被狂风切割粉碎却又瞬间挣扎着生长绽放。

震荡的狂风和巨大的咆哮声响彻整个平原，鲜血顺着纤细的手淌下，染红了莉姆的白裙，随风荡漾。

莉姆握紧巨大的法杖，看着前方。

凌冽的狂风，高昂的咆哮，与挣扎生长的蔷薇。

明明只过了一瞬间，却仿佛经历了无比漫长而艰难的持久战。

终于，伴着重物轰然倒塌的巨响，风翎枭龙庞大的身躯重重摔倒在草原上，无数细小的蔷薇从体表贯穿而出，于淡蓝色的绒羽上开满摇曳的红色花海。

结束了……

拢了拢被狂风掀起的长袍，夏伊侧身看向身后。

虽然地面早已狼藉一片，但好在车马没有实质性损伤。大部分的风刃被身穿铁甲的护卫挡了下来，在盾牌和铁甲上留下密密麻麻的刻痕。

“夏伊姐，快看快看，这可是我们‘蔷薇鸟’第一次成功狩猎亚龙种！”

夏伊的目光从战场收回，落在身旁的少女身上。莉姆正拄着那柄比她人还高的法杖，胸膛剧烈起伏着。

她的白裙下摆被龙血浸透，沉甸甸地垂着，裙角还在往下滴落暗红色的液滴，两侧束起的淡灰头发也差点被风吹散。

“嗯嗯，很厉害呀。”看着这个比自己矮半个头的少女，夏伊忍不住摸了摸她的头。

“嘿嘿——啊，姐姐来了。”

莉娅的身影从渐渐散去的血雾中浮现。她的步伐还算稳，但握枪的手在微微发抖，枪尖拖在身后，在草原上犁出一道浅浅的沟。走到近前，她把枪往地上一顿，整个人靠在枪杆上，大口喘气。

“呼——，猎人匕首呢，你不是来之前就嚷嚷要去剥取素材吗？”

莉娅笑容一僵，慌慌张张地从身后抽出了猎人匕首，然后拽着夏伊的袍子往前跑。

“咳咳，刚刚在讲解什么是素材，应该……”

“走吧，夏伊姐，我带你去剥取素材。”

夏伊跌跌撞撞地跟在莉姆身后，

“诶？我也要剥取吗？”

“对。”

莉姆不由分说地拽着夏伊往前跑，风翎枭龙的尸骸静静卧在前方，像一座倾倒的小山。淡青蓝色的绒羽在阳光下泛着柔和的光泽，但那之上，无数猩红的蔷薇从鳞片间穿刺而出，摇曳生姿。

“先从翅膀开始！”莉姆把法杖塞到夏伊的手里，然后自己跑到巨大的翼膜旁，匕首在指尖转了个漂亮的刀花，“翼膜边缘这些透明的风感翎羽，是制作探测设备的最佳材料——唔，应该从这里下刀……”

她的话音未落，夏伊的动作突然顿住，一种诡异的气息攥紧了夏伊的心脏。

腐败？

“莉姆——”

她的声音和风翎枭龙的咆哮声同时响起——不是风啸，不是龙吟，而是一种更古老、更威严的语言——龙语。

天穹之幕。

然后，狂风再起。不是从风翎枭龙为中心释放，而是从四面八方涌来，气流变得狂乱无序，以风翎枭龙为中心，形成一个直径超过百米的巨大漩涡，漩涡的边缘，空气凝成实质，化作一道道旋转的风墙，将这方天地彻底封闭。

铁甲护卫们还没来得及反应，就被无形的风压得跪倒在地。他们的铁甲发出令人牙酸的嘎吱声，盾牌上的刻痕在瞬间被无数道的风刃加深。马车也剧烈摇晃，木板发出濒临破碎的哀鸣。赤鬃马嘶鸣着跪倒，它们的鬃毛被风撕扯得四散飞扬。

莉娅的身形在狂风中踉跄，她试图冲向妹妹和夏伊所在的方向，但四周无数道杂乱无章的细密风刃将她死死压制在原地。

不知何时，风翎枭龙已张开双翼，飞至空中，睁开的金黄竖瞳内染上一抹墨色。

接着，俯冲而下——

风翎枭龙的利爪撕裂空气，距离莉姆不足三米，少女闭上了眼睛。

——轰！

地面碎裂的声音和一声沉闷的撞击。

不是利爪撕裂血肉的声音，而是某种更沉闷、更厚重的撞击。

莉姆睁开眼睛，夏伊冲到她面前，用巨大的法杖将利爪向下一拨，接着她像一片被狂风吹起的羽毛，顺着冲击的方向向后飘起，脚尖在风翎枭龙的爪尖一踩，身形在半空中划出一道不可思议的弧线，竟然跃上了风翎枭龙的头顶。

赤色的瞳孔中，夏伊看见了风翎枭龙头顶的蔷薇早已变得凋零漆黑——

她双手握杖，高高举起，法杖贯穿而下。

风翎枭龙发出一声凄厉的嘶鸣，接着一道道漆黑的雾气从伤口中被吸引出，像活着般扭曲着想要逃窜，却最终融入夏伊胸前的吊坠。

视角拉到远处，黑袍少女正抬起的手缓缓放下，她双眼有些不可置信的睁大。

“腐败……被净化了？”

风停了。

天穹之幕的乱流在一瞬间溃散，唯有轻风托着风翎枭龙的躯体，黄昏洒落，照在缓缓坠落的巨大身躯上。它的瞳孔恢复了澄澈的金黄色，在彻底失去意识的前一刻，它发出一声微弱的，轻轻的鸣叫。

怨恨……还是感谢？

然后，庞大的身躯轰然坠地。

夏伊从龙首上跳下，踉跄了一下，扶住身边的龙角才站稳。她的长袍在刚刚的乱流中又被割的残破不堪，胸前的吊坠还在微微发烫。

紧接着一个软乎乎的生物“唰”地一下扑到她身上，抱着她转了一圈。

“呜哇，夏伊姐！”

“好啦好啦，没事了。”

夏伊看着眼前这位少女抱着自己不肯松手，头还在自己怀里蹭来蹭去的。

莫名有种熟悉感和亲近感。

“我本来就是想着抱一下就好了，真的，只是夏伊姐的身体又软又暖和……”

“莉姆！”

莉娅的喊声刚从前方传来，莉姆就像一只受惊的小松鼠一样跳了起来，正打算绕到夏伊身后就被莉娅提着衣服拉到后面。

“……谢谢你救了莉姆。”眼前的少女虽然语气平淡，但格外郑重地抬起右手，拇指与小指轻轻相触，其余三指并拢向上，按于心口，躬身向夏伊致谢。

莉娅语气不变地说道：“一直待在这很危险，即使现在是染血草原的旭风季也一样。”

她顿了顿，继续说道：“因此，我代表猎人小队“蔷薇鸟”邀请你与我们同行。”

染血草原？旭风季？总之，先跟着走吧。

“嗯……可以呀。”
