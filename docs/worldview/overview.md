# 世界观总览

> *"高天之上的神明啊，我向您祈盼。愿您能佑我不度困苦、不失所愿……"*
> ——序

## 世界概览

这是一个龙族尚存、众神已逝、腐败在每一缕风中低语的世界。

龙族群岛——原初之伤，灰雾第一次被逼退的地方，也是它回潮压力最大的裂口。三位魔女前赴后继，正是因为这处伤口从未愈合。

## 龙族谱系

### 真龙种

祂们强大，祂们高贵，祂们是宛若神明般的存在。每一位达到"君王"位阶的真龙，其存在本身就是一个**文明奇点**。祂的呼吸改变气候，祂的意志塑造法律，祂的鳞片纹路化为建筑与文字。

真龙种按力量分为三个等阶：
- **神阶**——凌驾于君王之上的存在，文明的终结者
- **君王阶**——一国之君，文明的总根源
- **君主阶**——谱系的源头，沉睡于大地之中

### 亚龙种

无论是翱翔于天际的云鲸，还是深潜于渊底的蛟龙，甚至是以脊背为山脉的大陆龙；从宛若神明改变生态的高阶，到能被猎人们组团狩猎的低阶——它们皆为亚龙种。

- **鸟龙目**：风、苍穹与部分真龙力量的次级投影，广泛分布于大气至林冠等空域生态位。
- **兽龙目**："别把它们当猎物，当成一场天灾。"
- **飞龙目**："它们是天空的君王，移动的天灾，不朽的史诗。"
- **海龙目**：大得离谱、怪得吓人，能把船当点心吞了的神话级海怪。
- **空龙目**："那无垠的蓝，那流转的白——那是另一片国土。"

### 龙兽

龙族演化树上的旁支，智力低下，无龙语魔法，仅凭本能生存。它们构成了食物链的基座与中层。

## 魔法体系

在这个世界里，**所有武器都可以被视为法杖**——只是表现形式不同。

| 魔法体系 | 类型 | 说明 |
|:--------|:----|:------|
| 龙语魔法 | 血脉·先天 | 铭刻于血脉的古老记忆，如呼吸般本能 |
| ├─ 血脉魔法 | 血脉·传承 | 家族血脉中传承的力量 |
| └─ 民俗魔法 | 血脉·日常 | 融入日常生活的朴素法术 |
| 符文魔法 | 认知·后天 | 后天的科学，打破了血统垄断 |
| 仪式魔法 | 信仰·祈求 | 高天之上的神明对祈求之人的垂青 |

## 核心概念：龙权文明论

每一位"君王"阶真龙，其存在本身就是一个**文明奇点**。祂不仅是统治者，更是活着的图腾、行走的法则、文化的总根源。

- **被动辐射**——龙君庞大的魔力与独特的生命场域，会永久性改变领地内的生态环境与物理规则
- **主动塑造**——龙君的意志、审美与记忆，会通过龙语魔法、血脉赐福等方式灌输给子民
- **文明载体**——龙君本身的躯体形态、鳞甲纹路，都可能衍生出建筑风格与宗教仪轨

### 曦日君王 · 永昼王庭
- **文化核心**：秩序、辉煌、等级、外在的仪式美
- **现状**：君王陷入长眠，徒留腐朽的框架——文明失去了光源

### 黄昏君王 · 永暮国度
- **文化核心**：悲怆、永恒、循环、内在的静默美
- **现状**：被永恒隔绝于现世之外
