# 原初魔女与创世之伤

> *"当星星坠落于荒野，当真龙泯于神话，当腐蚀攀上意识——你听，那是群龙的怒吼。"*

## 伊始：灰雾中的降临

在无边灰雾弥漫之际，在未有神智的巨兽咆哮之际，龙族的故土上降临了第一位**原初魔女**。

然而魔女的身躯被龙族分食殆尽。吞噬祂血脉的龙族一部分成为了**真龙种**，得以净化；另一部分则受到腐蚀，日后成为了各式各样的**亚龙种**。

原初魔女的身躯融入地壳，于此地滋生了魔法，驱赶了灰雾，创造出生态。祂的伊始形成龙族浮岛四周密布的粒子风暴。

此时，龙族仍然只是拥有庞大力量的野兽，以厮杀为生。

## 第一纪元：人创时代

在这原始之地，龙与龙杀戮与共存的世界中，降下了**第二位魔女**。

祂以龙血与自身的血脉，创造了人。一些人类成为龙族祭司，创造了一个又一个国家，人类成为龙族文明的代言者。他们信仰着真龙。

然而原初魔女诞生的**腐蚀**日趋严重……

## 第二纪元：神战时代

由于真龙种间的文明冲突和腐败影响，诸神混战。第二位魔女也陷入沉眠，并赐下预言。最终以"神"的诞生终止了战争。

## 第三纪元：腐蚀纪元

真龙也逐渐受到了腐蚀。夏伊斯岛的君王**露曦莉娅**想要用毁灭一切、重启来清除污染。

直到预言中第三位魔女降临……

她会面对什么？带来什么呢？

*——挑战！净化！共存！*
